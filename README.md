# 氢·简 / Hydro-Minim

Hydro-Minim is the first Halo theme in the Hydro series. It keeps the original Hydrogen-inspired visual language while using Halo native Thymeleaf templates and Vite-powered TypeScript interactions.

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
